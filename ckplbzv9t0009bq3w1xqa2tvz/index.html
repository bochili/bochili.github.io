如何卸载Kali Linux

0x00

卸载Grub——

这里我看了Kali中文网上的一个帖子

地址：http://www.kali.org.cn/thread-11517-1-1.html

他这里是用Mbrfix工具来删除Grub，然后我按照他的步骤：

  1. 将软件放到C盘根目录下
  2. 打开CMD，执行命令：
    cd \ mbrfix /drive 0 fixmbr /yes
  3. 重新启动电脑
    然后我重启了，遇到如下情况：

Error Loading Operating System

于是我临时给U盘做了PE系统，修复MBR，死活进不去Windows系统。并且没有“Error Loading Operating System”这个提示了，反而出现了“Couldn’t find NTLDR”

然后上网找了找方法：

于是我又进入Dos版本的DiskGenius（U深度装机自带，不是打广告QAQ）

如下图所示（事发时心情过于激动，忘记拍照了，这张图片是现在临时截的），进入Dos版本的DiskGenius后，右键系统盘，选择“重建主引导记录(MBR)”，根据提示重建即可。

然后重新启动电脑，会发现Windows正常启动。

0x01

删除分区——

进入Windows自带的分区工具，找到之前安装Kali Linux系统的分区，右键，删除即可

Ending

至此，关于Kali Linux卸载过程中遇到的问题就解决了，下面是卸载过程中用到工具的下载地址：

MbrFix.exe：https://cdn.jsdelivr.net/gh/bochili/fontscdn/MbrFix.exe

DiskGenius：http://tcy.mqego.com/diskgenius_pro.zip
