如何卸载Kali Linux

Posted at 2018-06-01 引导 

0x00

卸载 Grub——

这里我看了 Kali 中文网上的一个帖子

地址：http://www.kali.org.cn/thread-11517-1-1.html

他这里是用 Mbrfix 工具来删除 Grub，然后我按照他的步骤：

  1. 将软件放到 C 盘根目录下
  2. 打开 CMD，执行命令：
    cd \ mbrfix /drive 0 fixmbr /yes
  3. 重新启动电脑
    然后我重启了，遇到如下情况：

Error Loading Operating System

于是我临时给 U 盘做了 PE 系统，修复 MBR，死活进不去 Windows 系统。并且没有“Error Loading Operating System”这个提示了，反而出现了“Couldn’t find NTLDR”

然后上网找了找方法：

于是我又进入 Dos 版本的 DiskGenius（U 深度装机自带，不是打广告 QAQ）

如下图所示（事发时心情过于激动，忘记拍照了，这张图片是现在临时截的），进入 Dos 版本的 DiskGenius 后，右键系统盘，选择“重建主引导记录(MBR)”，根据提示重建即可。

然后重新启动电脑，会发现 Windows 正常启动。

0x01

删除分区——

进入 Windows 自带的分区工具，找到之前安装 Kali Linux 系统的分区，右键，删除即可

Ending

至此，关于 Kali Linux 卸载过程中遇到的问题就解决了，下面是卸载过程中用到工具的下载地址：

MbrFix.exe：https://cdn.jsdelivr.net/gh/bochili/fontscdn/MbrFix.exe

DiskGenius：http://tcy.mqego.com/diskgenius_pro.zip
