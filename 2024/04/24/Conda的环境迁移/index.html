Conda的环境迁移

Posted at 2024-04-24 瞎折腾  Python 

原环境中导出

在已经激活 Conda 环境的终端中执行：

conda env export > environment.yml

此时当前目录下会生成一个 environment.yml ，将此文件拷贝到目标机器中。

在目标机器中部署

conda env create -f environment.yml

如果没有 conda 的环境变量，需要找到 conda 所在的路径，执行：

/xxx/conda.bat env create -f environment.yml
