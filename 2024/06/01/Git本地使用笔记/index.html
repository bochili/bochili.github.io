git init 初始化当前目录为git仓库，完成后会在该目录下新建一个.git目录
git status 查看当前暂存区情况
在vscode中的git界面点击文件右边的加号将文件添加到暂存区 命令为 git add 文件名 支持通配符
.gitignore 用于忽略文件 如*.exe 会忽略所有以.exe为后缀的文件

使用git rm --cached 文件名 将文件从暂存区删除
git commit -m "内容" 提交暂存区内容
git log 查看提交记录
git branch 查看仓库的所有分支

git branch 分支名称 创建新分支但不切换
git checkout -b 分支名称 创建新分支并切换到该分支
git switch 分支名称 切换到指定分支
git merge 分支名称 将指定分支合并到当前所在分支

git revert 将当前节点与某个指定的祖先节点进行合并
git revert COMMIT_ID 将当前节点与指定祖先节点合并，一般会出现冲突 需要手动编辑文件解决冲突
git revert --continue 冲突解决后继续合并
