C基础

STDINT.H 头文件

位于 MDK\ARM\ARMCC\include 目录下，源自 C99 标准。其中定义了 uint_8 这些类型别名。

使用该头文件需要在 MDK 中开启 C99 支持（默认开启），选项在 Options > C/C++ > C99 Mode

位运算

运算符 含义
& 按位与
| 按位或
^ 按位异或
~ 按位取反
<< 左移
>> 右移 
按位与：两个都是1才是1
0 & 0 = 0
1 & 0 = 0
1 & 1 = 1
按位或：有一个是1就是1
1 | 0 = 1
0 | 0 = 0
1 | 1 = 1
按位异或：两个不一样才是1，一样为0
1 ^ 1 = 0
0 ^ 0 = 0
1 ^ 0 = 1
0 ^ 1 = 1
按位取反：把每一位上的数字取反
~11100100 = 00011011
左移：八位二进制数，左移几位，就在右边添几个零，左边多出来的去掉。右移则反之
11100100 << 2 = 10010000
11100100 >> 2 = 00111001
