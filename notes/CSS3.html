盒子定义

在使用 padding 设置内边距时，会出现盒子被padding撑大的效果，为了避免这种现象，我们将该盒子进行定义：

box-sizing: border-box;

一般在开发时，直接为 * 选择器添加该样式：

* {
    box-sizing: border-box;
}

背景图片

基础设置

background-image: url(xxx.jpg);

行级元素，如 span ，在浮动后将会自动转换为块级元素，允许设置宽高等属性。

清除浮动

下面这个例子，两个方块分别左右浮动，其父级元素就无法感知它们的宽高，我们需要一个元素把父级元素撑起来。

<div class="main">
    <div class="div1"></div>
    <div class="div2"></div>
</div>

clear 属性

div{
    width:200px;
    height:200px;
}
div.red{
    background:red;
    float:left;
}
div.green{
    background:green;
    float:right;
}
div.blue{
    background:blue;
    clear: both | left | right;  
    通过清除浮动，可以消除上面浮动元素对该元素空间位置的影响，也就是说blue这个div的位置不会动
}

方法一：定义 clearfix 类

上面介绍了 clear 方法，那么我们可以在这俩子级div下面添加一个元素，令其清除上方的浮动，就可以把 main 盒子撑起来了：

<div class="main">
      <div class="div1"></div>
      <div class="div2"></div>
      <div class="clearfix"></div>
</div>

.clearfix {
  clear: both;
}

![[Pasted image 20221129182249.png]]

方法二：使用after伪类清除浮动

借鉴上面的方法，我们可以给盒子添加一个after 伪类，让这个伪类去清除浮动：

.div2::after{
    content:"";
    clear: both;
    display:block;  注意伪类本身不是块级元素，需要将其转换为块级元素
}

方法三：CSS的BFC机制

可以给.main 类添加一个overflow属性，就会触发BFC机制，就可以直接将其撑起来：
.main{
  width: 240px;
  padding: 10px;
  border: 2px solid #000;
  overflow: auto;
}

形状及文字环绕形状

span.shape {
  width: 100px;
  height: 100px;
  float: left;
  padding: 30px;
  border: 5px solid red;
  margin: 10px;
  clip-path:circle(50%);   // 修改成大小为50%的圆形
  clip-path: circle(50% at top right);  // 圆心位于右上角
  clip-path: ellipse(50% 30%);   // 椭圆形
  clip-path: polygon(50% 0,100% 100%,0 100%); // 绘制多边形，参数是坐标点位置x y
  shape-outside: polygon(50% 0,100% 100%,0 100%);  令shape-outside与clip-path相同，文字就可以沿着形状进行环绕了
}
