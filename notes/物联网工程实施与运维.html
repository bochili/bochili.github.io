任务八：Windows 2019操作系统管理（10分）

1****、Windows 2019操作系统管理（10分）

Ø 任务描述：为防止服务器出现宕机，需要对服务器进行运行监控，要求在Windows 2019系统下使用命令提示符查询当前服务器磁盘使用情况、内存使用情况、CPU占用率、网卡使用情况。

Ø 任务要求：

(1) 利用命令提示符查询磁盘使用情况。将查询过程及结果****截全屏粘贴至“work\Result.docx”的任务八中；

(2) 利用命令提示符查询内存使用情况。将查询过程及结果****截全屏粘贴至“work\Result.docx”的任务八中；

(3) 利用命令提示符查询CPU占用率。将查询过程及结果****截全屏粘贴至“work\Result.docx”的任务八中；

(4) 利用命令提示符查询网卡使用情况。将查询过程及结果****截全屏粘贴至“work\Result.docx”的任务八中。

# 查看磁盘
diskpart
list disk  # 查看每个磁盘使用情况
list vol # 查看每个分区的使用情况
CTRL+C 退出DISKPART

# 查看内存情况
systeminfo
#其中包括如下内容：
Total Physical Memory:     15,741 MB  # 物理内存大小
Available Physical Memory: 7,443 MB   # 剩余可用内存
Virtual Memory: Max Size:  19,453 MB  # 最大虚拟内存
Virtual Memory: Available: 7,884 MB   # 剩余可用的虚拟内存
Virtual Memory: In Use:    11,569 MB  # 目前占用的虚拟内存

# 查看CPU占用率
wmic cpu get loadpercentage

# 查看网卡使用情况：
# 1：
systeminfo # systeminfo下的Network Card(s)展示了每张网卡的型号、连接名、是否开启DHCP、DHCP服务器IP地址、和本机的IPV4和IPV6地址
# 2:
ipconfig # 可以查看每张网卡的IP配置信息

任务九：网络协议客户端发送数据并在设备查看遥测值（5分）

1****、网络协议客户端配置（3分）

Ø 任务描述：NL公司的售后工程师近期收到客户反馈，描述为：云平台不能正常显示温湿度值。售后工程师根据多年经验，怀疑是云平台的MQTT通信出现故障。于是，决定使用MQTT.fx软件模拟MQTT客户端向云平台发送测试数据尝试排除故障。

Ø 任务要求：

(1) 安装考试资源包中的MQTT.fx软件；

(2) 云平台地址：52.131.248.66、MQTT端口号：1883；

(3) 订阅主题为：v1/devices/me/telemetry；

(4) 配置结果****截全屏并粘贴至“work\Result.docx”的任务九中。

2****、遥测值查看（2分）

Ø 任务描述：同上。

Ø 任务要求：

(1) 发送键为Light,值为：考号的测试数据。如考号为99，则发送Light,值为：99；

(2) 云平台设备名称为MQTT+考号，如考号为99，则设备名称为MQTT99；

(3) 将云平台遥测数据****截全屏并粘贴至“work\Result.docx”的任务九中。
