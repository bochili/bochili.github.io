Vue3.0在组件内this.$route.path一直为"/"的解决办法

Posted at 2022-05-22 解决问题  前端 vue 

watch: {
    $route(val){
      console.log(val);
    }
  }

val 是一个对象类型 其中有一个fullPath

即 val.fullPath 即可拿到最新的当前路径

对 Watch 函数的解释

Watch 意为监视，当页面中的响应式 Data 发生变化时，将会触发 Watch 中的函数。

上面代码中，我监听了 this.$route 的值，该数据发生变化，则会将参数输出到控制台。
